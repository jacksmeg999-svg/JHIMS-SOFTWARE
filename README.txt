DMH / JHIMS Hospital Web System (Embedded app.py version)

How to run:
1. Install Python 3.
2. Install dependencies: pip install -r requirements.txt
3. Run the app: python app.py
4. Open http://127.0.0.1:5000 in your browser.

Default logins:
- admin / admin123
- cashier / cashier123
- lab / lab123
- mortuary / mortuary123
- reports / reports123

Replace static/jhims_logo.png and static/ghs_logo.png with your real logos (PNG).
